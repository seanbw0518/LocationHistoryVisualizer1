## README ##

### Visualizer 1:
This is before Google changed its Timeline format. It's more accurate, but more out of date.
1. Put `Records.json` in this folder. This contains all your Google location history.
2. Run `visualizer.py`.
3. The interactive map is saved to `map.html`, open this with a browser to view the map!

### Visualizer 2:
This is since Google changed its Timeline format. It's less accurate, but more up to date.
1. Put `Timeline.json` in this folder. This contains all your Google location history.
2. Run `visualizer 2.py`.
3. The interactive map is saved to `map-new.html`, open this with a browser to view the map!
